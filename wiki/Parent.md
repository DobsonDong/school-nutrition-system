# 家长端功能说明

家长端采用移动端适配设计（max-width: 480px），面向学生家长提供食谱查看、点餐、评价和营养报告服务。

## 页面列表

### 1. 首页 (`index.html`)

家长端主入口，汇聚核心信息和快捷操作。

**核心功能：**
- 学生信息头部（学校、班级、姓名）
- 营养提醒预警卡片
- 今日午餐菜单详情（菜品名、热量、蛋白质、克重）
- 快捷入口：点餐、评价、报告、溯源
- 本周营养摄入概要（能量、蛋白质、达标率）
- 满意度评价入口
- 本周食谱预览（周一至周五缩略）
- 底部导航栏：首页、点餐、评价、报告

### 2. 在线点餐 (`order.html`)

支持家长为孩子选择每日餐食。

**核心功能：**
- 日期选择器（周一至周五）
- 餐别切换（早餐/午餐/晚餐）
- 营养标签筛选：高蛋白、低热量、高钙、素食
- 推荐套餐：A/B套餐，含菜品组合、营养汇总、价格
- 单点菜品：菜品卡片含图标、描述、营养标签、价格、加减数量
- 购物车栏：已选数量、总价、营养汇总、去结算

### 3. 满意度评价 (`satisfaction.html`)

家长对每日餐食的多维度评价。

**评价维度：**
- 口味满意度（5星 + 标签：味道好/偏辣/偏咸/偏甜/刚刚好）
- 份量满意度（5星 + 标签：份量足/刚刚好/有点少）
- 温度满意度（5星 + 标签：热乎乎/温度适中/有点凉）
- 卫生满意度（5星 + 标签：很干净/一般/需改进）
- 意见建议文本框 + 语音输入
- 提交成功弹窗

### 4. 营养摄入报告 (`nutrition-report.html`)

学生个人营养摄入的综合分析报告。

**核心功能：**
- 周期切换：周报/月报/学期报
- 膳食健康综合评分（百分制）
- 营养素摄入网格：能量、蛋白质、脂肪、碳水、钙、铁（达标/偏低状态）
- 近30天能量摄入趋势柱状图
- 与同龄人对比表（营养素、您的孩子 vs 同龄人平均、差异百分比）
- AI个性化建议（增加钙摄入、补充维C、运动营养建议）
- 膳食结构环形图 + 分项评分（主食搭配、蛋白质、蔬果摄入、多样性）

### 5. 食材溯源 (`traceability.html`)

查看每日食材的全链路溯源信息。

**核心功能：**
- 扫码查溯源入口
- 食材搜索（名称/批次号）
- 食材安全检测合格标识
- 基本信息：食材、供应商、产地、配送日期、保质期
- 溯源链路时间线：
  - 学校食堂烹饪（厨师、烹饪方式、出餐温度）
  - 入库检测（检测员、温度）
  - 冷链配送（车辆编号、全程温度、运输距离）
  - 屠宰加工（工厂、检疫证号）
  - 养殖基地（产地、养殖周期、养殖方式）
- 检测资质证照：动物检疫合格证、食品经营许可证、农药残留检测、兽药残留检测
- 今日其他食材溯源快捷入口

### 6. 完整周食谱 (`weekly-recipe.html`)

展示完整的一周食谱详情。

**核心功能：**
- 周次标题和日期范围
- 上一周/下一周切换
- 日期Tab切换（周一至周五，标记今天）
- 当日营养汇总（能量、蛋白质、脂肪）
- 午餐菜品详情列表（图标、菜名、营养数据、克重）
- 本周各日营养对比表（能量、蛋白质、脂肪、达标状态）
- 本周菜品矩阵一览（5x4 网格：主食/荤菜/素菜/汤羹 x 周一至周五）
